**Generic Security Playbooks**
This repository stores generic scripts, rules, playbooks, and a reference for tools in use for detection engineering. While a work-in-progress, rules and scripts are meant to be generically applied to assist in detection/mitigation efforts. Currently, playbooks include parsed TTPs gathered from CISA advisories from their #StopRansomware initiative. Users are responsible for their own security, management, and implementation of their environment. 

**Usage**
All PowerShell scripts can be found under _scripts/win_powershell_. Rules can be found under _rules_ in YAML format consistent with SIGMA standards. Playbooks can be found under _playbooks/ransomware_ and imported into CISA's Eviction Strategies tool found at https://www.cisa.gov/eviction-strategies-tool.

The _tools_ directory includes a list of links for certain tasks including analysis, system configuration, and detection toolsets.

Clone the repository to store it on your local system or use as a reference guide through the GitHub site.
